Equalities.

When we speak of an equality, we are talking about two mathematical statements which are equal to one another. This can be used in many ways whether by trying to simplify equations or solving for specific variables. The example shown in the images is an equality being used to find the value of x. This is done by isolating x to one side via subtracting like terms, getting the numeric values to the other side via subtraction, and finally dividing to get the value of x.

Inequalities

Despite its name inequalities aren't too different from equalities. Inequalities represent two mathematical statements which have some fixed relation to one another. These relations normally come in the form of greater than, less than, greater or equal to, and less than or equal to. They can be resolved in much the same way that equalities can with equal actions taken on both sides.
